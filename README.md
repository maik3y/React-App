# perfection-tv

Website for perfection.tv

# quick start

Install dependencies.

```
yarn
```

Run in development.

```
yarn start
```

# production

Run the `build` command to create a build in the `dist` folder.

```
yarn build
```

# cleanup

To get rid of all the autogenerated files by this project, run the `clean` command.

```
yarn clean
```
